## Features

* Create new tasks
* Mark tasks as completed
* Deleting tasks
* Updating your tasks
* Filter tasks by status (All, Completed, Non Completed)
* Change the order of tasks (Newest, oldest)
* Change themes: light dark
* Responsive Design
